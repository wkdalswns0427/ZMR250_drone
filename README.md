# ZMR250_drone
zmr-250 size drone
process by Arduino Nano
MPU-6050 with AR620-DX6e transmitter

- MPU6050
- BL Heli 12A ESCs
- CW/CCW BLDC Motors
- 3-cell LiPO battery
- carbon frame
- FC on Arduino Nano(32, old bootloader)

** should upload after selecting 'Old Bootloader' on Arduino IDE's Tool **
